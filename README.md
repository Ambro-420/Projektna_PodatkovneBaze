# SPLETNA STRAN ZA TRGOVCE
### avtorja: Klemen Žukovec, Ambrož Pleško
Namen projekta je narditi predlogo (template) stran na kateri bodo trgovci objavlali svoje prodtukte, njihovo ceno, sliko, opis in zalogo. Vsakemu trgovcu bi spletno stran prilagodili, vendar to je projket, ki pride v poštev izven študijskega okolja.
Funcija projekta je narediti stran kupcem čim bolj projazno in pregledno.
